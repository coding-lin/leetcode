### TS 篇

- node 环境下运行 ts

  1. 使用 `npm init -y` 初始化 `package.json`
  
  2. 安装依赖

  ```
  npm i @types/node --save-dev
  npm i ts-node -g
  npm i typescript --save-dev
  ```

  3. 运行指定 ts 文件 `ts-node 1.ts`
